# 爬虫蓝图系统功能清单

## 设计理念

这是一个**可视化编程系统**，类似 Unreal Engine 蓝图，通过拖拽节点和连线来构建爬虫流程。核心目标是将爬取的数据**最终写入数据库**。

状态说明：✅ 已实现，⏳ 未实现。

节点 JSON 可选支持 `remark` 字段用于节点注释说明；如果需要画布上的视觉批注，也可以使用 `comment` 类节点承载。

## 现代框架输入处理

### Vue 组件设置值

```javascript
// 方法1: 通过 v-model 的 __vModel 属性
const el = document.querySelector(selector);
if (el.__vModel) {
  el.__vModel.value = newValue;
}

// 方法2: 触发 input 和 change 事件
el.value = newValue;
el.dispatchEvent(new Event('input', { bubbles: true }));
el.dispatchEvent(new Event('change', { bubbles: true }));

// 方法3: 找到 Vue 实例
const vueInstance = el.__vueParentComponent?.proxy || el.__vue__;
if (vueInstance) {
  vueInstance.someProperty = newValue;
}
```

### React 组件设置值

```javascript
// 方法1: 通过 React 内部状态
const setReactValue = (el, value) => {
  const valueSetter = Object.getOwnPropertyDescriptor(el, 'value').set;
  const prototype = Object.getPrototypeOf(el);
  const prototypeValueSetter = Object.getOwnPropertyDescriptor(prototype, 'value').set;

  if (valueSetter && valueSetter !== prototypeValueSetter) {
    prototypeValueSetter.call(el, value);
  } else {
    valueSetter.call(el, value);
  }

  el.dispatchEvent(new Event('input', { bubbles: true }));
};
```

### Playwright 中的实现

```rust
// Rust + Playwright 示例
let element = page.wait_for_selector(selector).await?;

// 智能填充
element.fill(text).await?;  // Playwright 已经处理了大部分框架

// 或者强制触发事件
page.eval_on_selector(selector, r#"
  (el, value) => {
    el.value = value;
    el.dispatchEvent(new Event('input', { bubbles: true }));
    el.dispatchEvent(new Event('change', { bubbles: true }));
  }
"#, serde_json::json!([text])).await?;
```
